###  RideNow — Ride-Hailing Web App
> MERN · Socket.IO · Leaflet.js · GeoApify API · JWT

A full-stack ride-hailing platform replicating core Uber flows — booking, driver assignment, and live tracking.

- JWT-protected REST endpoints with role-based access (rider / driver / admin)
- Bidirectional Socket.IO layer keeping rider and driver views in sync across concurrent sessions
- Third-party geolocation (GeoApify + Leaflet.js) with graceful API failure handling

> MERN · Redux · TanStack Query & Router · JWT · Docker Compose
